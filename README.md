# Library Management System - Bharat Institute of Technology

A modern, feature-rich web-based library management system designed for Bharat Institute of Technology. Built with vanilla HTML, CSS, and JavaScript for maximum portability and ease of deployment.

## 🌟 Features

### For Students
- **Browse Books**: Search and filter through the entire book catalog
- **Borrow Books**: Easy one-click borrowing system with automatic due date calculation
- **Track Borrowed Books**: View all currently borrowed books with due dates
- **Fine Calculator**: Automatic fine calculation for overdue books (₹5 per day)
- **Dashboard Statistics**: Quick overview of borrowed books, available books, and total fines

### For Administrators
- **Book Management**: Add, edit, and delete books from the catalog
- **Transaction History**: View all borrowing and return transactions
- **Statistics Dashboard**: Real-time stats on total books, borrowed books, and active students
- **Search & Filter**: Advanced search by title, author, ISBN, or book ID
- **Category Management**: Organize books by categories

## 🎨 Design Features

- **Premium Dark Theme**: Modern dark mode with vibrant gradients
- **Glassmorphism Effects**: Sleek backdrop blur effects on cards
- **Smooth Animations**: Micro-interactions for enhanced user experience
- **Responsive Layout**: Works on desktop, tablet, and mobile devices
- **Accessibility**: Semantic HTML and proper form labels

## 🚀 Getting Started

### Installation

1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. No server setup required - runs entirely in the browser!

## 📚 How to Use

### As a Student

1. **Login** with student credentials
2. **Browse Books** in the catalog using search and filters
3. **Borrow a Book** by clicking the "Borrow Book" button
4. **View Your Books** in the "My Borrowed Books" section
5. **Return Books** when done - fines are calculated automatically for overdue items

### As an Administrator

1. **Login** with admin credentials
2. **View Dashboard** statistics for quick insights
3. **Add New Books** using the "Add Book" button
4. **Edit Books** by clicking the edit button on any book card
5. **Delete Books** using the delete button
6. **Monitor Transactions** in the Recent Transactions section

## 💡 Key Information

- **Borrowing Period**: 14 days from the date of borrowing
- **Fine Rate**: ₹5 per day after the due date
- **Data Storage**: Uses browser localStorage (data persists across sessions)
- **Sample Data**: Includes 10 pre-populated books across various categories
